# Student Marks Analyzer using NumPy

This project analyzes student marks using Python and NumPy.

## Features
- Calculate total, average, highest and lowest marks
- Perform analysis using NumPy arrays
- Simple and beginner-friendly

## Technologies Used
- Python
- NumPy
- Jupyter Notebook

## How to Run
1. Open the notebook file
2. Run all cells step by step
